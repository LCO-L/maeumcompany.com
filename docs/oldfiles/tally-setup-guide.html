⚡ Tally Form 빠른 설정

5분 안에 MAEUM 대기자 명단 Form 만들기

🎯 Step 1: Tally 계정 만들기 (30초)

Tally 가입하기 (무료)

이메일과 비밀번호만 입력하면 끝! (또는 Google 계정으로 가입)

📝 Step 2: Form 만들기 (2분)

로그인 후 "Create form" 클릭 → "Start from scratch" 선택

Form 제목 설정:

Form name: MAEUM - Join the AI Revolution

Description: Be the first to know when MAEUM launches in your region. Top-tier AI for everyone, everywhere.

필드 추가하기:

1. Email (Email 타입) - Required ✅ Label: Email Address Placeholder: your@email.com 2. Name (Short text) - Optional Label: Full Name Placeholder: John Doe 3. Organization (Short text) - Optional Label: Organization/Company Placeholder: Your company or institution 4. Country (Dropdown) - Required ✅ Label: Country/Region Options: - South Korea (한국) - United States - Japan - China - India - Other 5. Interest (Single choice) - Required ✅ Label: What's your primary interest in MAEUM? Options: - Government/Public Sector - Education - Healthcare - Enterprise/Business - Personal Use - Investment/Partnership 6. Message (Long text) - Optional Label: Message (Optional) Placeholder: Any questions or specific needs? 7. Newsletter (Single choice) - Required ✅ Label: Would you like to receive updates? Options: - Yes, keep me updated - No, just notify me at launch

🎨 Step 3: 디자인 설정 (30초)

  • 왼쪽 메뉴에서 "Design" 클릭
  • Color scheme: Blue 선택
  • Font: Inter 또는 Helvetica
  • Button style: Rounded
  • Submit button text: "Join the Movement"

🔗 Step 4: Form ID 가져오기 (10초)

  1. 상단의 "Share" 버튼 클릭
  2. "Embed" 탭 선택
  3. URL에서 Form ID 복사
https://tally.so/r/wA7dBe ← 이 부분이 Form ID 예시: wA7dBe, nWq9Qw, mVBgPo 등

📌 Form ID 확인하는 더 쉬운 방법:

Form 수정 화면 URL을 보세요:

https://tally.so/forms/wA7dBe/edit

✅ Step 5: index.html 업데이트 (30초)

Form ID를 복사했으면, index.html에서 이 부분을 수정:

생성된 코드:

Line 998에서 수정:

// 기존 const TALLY_FORM_ID = 'YOUR_TALLY_FORM_ID'; // 수정 후 (예시) const TALLY_FORM_ID = 'wA7dBe';

🎉 Step 6: 테스트 (10초)

  1. index.html을 브라우저에서 열기
  2. "Join the Movement" 버튼 클릭
  3. Tally Form이 팝업으로 열리는지 확인
  4. 테스트 제출 해보기

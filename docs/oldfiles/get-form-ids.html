📝 Entry ID 찾는 방법:

  1. Form 미리보기 클릭
  2. F12 (개발자 도구) 열기
  3. Console 탭에서 다음 코드 실행:
// 모든 entry ID를 찾는 코드 document.querySelectorAll('[name^="entry."]').forEach(el => { console.log(el.name, '→', el.placeholder || el.getAttribute('aria-label')); });

entry.1234567890 → Email Address
entry.987654321 → Full Name

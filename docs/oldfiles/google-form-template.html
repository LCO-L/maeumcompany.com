Step 2: Form 설정 (복사해서 사용)

📝 Form 제목과 설명

제목: MAEUM - Join the AI Revolution 설명: Be the first to know when MAEUM launches in your region. Top-tier AI for everyone, everywhere.

📧 필수 필드 - 이메일

질문: Email Address * 유형: 단답형 필수: ✅ 응답 확인: 이메일

🏢 선택 필드 - 조직

질문: Organization (Optional) 유형: 단답형 설명: Your company, government agency, or institution

🌍 선택 필드 - 국가

질문: Country/Region 유형: 드롭다운 옵션: - South Korea (한국) - United States - Japan - China - India - Indonesia - Philippines - Vietnam - Thailand - Malaysia - Singapore - Germany - France - United Kingdom - Brazil - Mexico - Nigeria - Kenya - South Africa - Other

💼 선택 필드 - 관심 분야

질문: Primary Interest 유형: 객관식 (라디오 버튼) 옵션: ○ Government/Public Sector (정부/공공) ○ Education (교육) ○ Healthcare (의료) ○ Enterprise/Business (기업) ○ NGO/Non-profit (NGO/비영리) ○ Personal Use (개인) ○ Investment/Partnership (투자/파트너십)

Step 3: Form ID 찾기

  1. Form 상단의 "보내기" 버튼 클릭
  2. 링크 아이콘 (🔗) 클릭
  3. URL 복사
  4. Form ID는 URL의 이 부분입니다:
https://docs.google.com/forms/d/e/1FAIpQLSf_이부분이_Form_ID/viewform

Step 4: Entry ID 찾기

  1. Form 미리보기 클릭
  2. F12 눌러서 개발자 도구 열기
  3. 이메일 입력 칸에서 우클릭 → 검사
  4. HTML에서 name="entry.XXXXXXX" 찾기
💡 Tip: Ctrl+F (또는 Cmd+F)로 "entry." 검색하면 쉽게 찾을 수 있습니다!
<input type="email" name="entry.1234567890" ...>

Step 5: 웹사이트 코드 업데이트

index.html 파일에서 다음 두 곳을 수정:

📍 Line 986 & 1132 근처

// 기존 코드 const GOOGLE_FORM_URL = 'https://docs.google.com/forms/d/e/YOUR_FORM_ID/formResponse'; const EMAIL_ENTRY_ID = 'entry.YOUR_ENTRY_ID'; // 실제 값으로 변경 (예시) const GOOGLE_FORM_URL = 'https://docs.google.com/forms/d/e/1FAIpQLSf_abc123xyz/formResponse'; const EMAIL_ENTRY_ID = 'entry.1234567890';

Step 6: 응답 스프레드시트 연결

  1. Form에서 "응답" 탭 클릭
  2. 스프레드시트 아이콘 (📊) 클릭
  3. "새 스프레드시트 만들기" 선택
  4. 이름: "MAEUM Waitlist"
✅ 완료! 이제 이메일이 자동으로 Google Sheets에 저장됩니다.

Step 7: 테스트

  1. 웹사이트에서 "Join the Movement" 버튼 클릭
  2. 테스트 이메일 입력 (예: test@example.com)
  3. 제출 후 Google Sheets 확인
  4. 데이터가 들어왔는지 확인

🎯 추가 설정 (선택사항)

이메일 알림 받기

  • Form → 응답 → ⋮ 메뉴 → "새 응답 이메일 받기"

자동 응답 이메일

  • 애드온 → "Email Notifications for Forms" 설치
  • 템플릿 설정으로 자동 환영 이메일 발송

스팸 방지

  • 설정 → 프레젠테이션 → "reCAPTCHA 필요" 활성화
